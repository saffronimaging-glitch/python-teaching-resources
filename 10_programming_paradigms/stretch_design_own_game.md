# Stretch Challenge - Design Your Own Game

## Task

Design your own small game using:
- decomposition
- hierarchy charts
- top-down design

---

# Your Game Must Include

- A welcome screen
- User input
- A scoring system
- At least 3 subroutines/functions

---

# Extension Tasks

- Add levels
- Add multiple question categories
- Add a timer
- Add random questions

---

# Planning Task

Create:
- a hierarchy chart
- a structure chart
- a list of subroutines
