# Hierarchy Chart - Quiz Game

```text
Quiz Game
│
├── Welcome Screen
│
├── Question System
│   ├── Ask Question
│   ├── Check Answer
│   └── Display Feedback
│
├── Score System
│   ├── Add Score
│   └── Display Final Score
│
└── End Screen
```

---

# Purpose of a Hierarchy Chart

Hierarchy charts:
- break systems into modules
- show relationships between components
- support top-down design
