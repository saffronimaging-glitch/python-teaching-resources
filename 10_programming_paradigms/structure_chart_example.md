# Structure Chart Example

## Procedural Quiz Program

```text
main()
│
├── welcome_message()
│
├── ask_question()
│
├── check_answer()
│
├── update_score()
│
└── display_final_score()
```

---

# Why Use Structure Charts?

Structure charts:
- show subroutine relationships
- help organise procedural programs
- improve modularity
