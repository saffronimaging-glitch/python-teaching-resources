# Decomposition Example

## Problem
Create a simple quiz game.

---

## Breaking the problem into smaller tasks

### Main Quiz System
- Display welcome message
- Ask questions
- Check answers
- Keep score
- Display final result

---

## Question System
- Store question
- Store answer
- Check user input

---

## Score System
- Add points
- Display score

---

# Why Use Decomposition?

Decomposition:
- makes problems easier to manage
- helps programmers organise code
- allows teams to work on separate parts
- improves debugging
